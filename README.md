# 📝 Project 1 – To-Do List Application

This project was developed as part of my internship at DecodeLabs.

# 📌 Project Overview

A simple and interactive To-Do List application developed using Python. The program allows users to add, view, and manage tasks efficiently through a console-based interface.

# 🚀 Features

* Add new tasks
* View all tasks
* Delete completed tasks
* Save tasks to a file
* Load saved tasks automatically
* User-friendly menu-driven interface

# 🛠 Technologies Used

* Python
* File Handling
* Lists
* Loops
* Functions
* Exception Handling

# 📚 Learning Outcomes

This project enhanced my understanding of Python data structures, file handling, CRUD operations, and menu-driven program development.
